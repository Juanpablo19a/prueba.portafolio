Mi Portafolio

Aquí encontrarás algunos de los proyectos que he desarrollado durante mi formación como desarrollador web. Cada uno refleja lo que estoy aprendiendo y cómo aplico tecnologías modernas en soluciones reales.

Sistema de Reservas

Sistema web de reservas que permite crear y cancelar reservas de forma dinámica, con persistencia de datos mediante LocalStorage, sin necesidad de backend.

Tecnologías: JavaScript, HTML5, CSS3, LocalStorage

Buscador de GIFs

Aplicación web desarrollada con React y TypeScript que consume la API de Giphy para buscar y mostrar GIFs en tiempo real. Incluye historial de búsquedas persistente en LocalStorage y una interfaz responsiva.

Tecnologías: React, TypeScript, API de Giphy, HTML5, CSS3, LocalStorage

Rick and Morty App

Aplicación web desarrollada con React utilizando Create React App que consume una API pública para mostrar información de personajes de Rick and Morty, aplicando renderizado dinámico y buenas prácticas de construcción.

Tecnologías: React, JavaScript, Create React App, API REST, HTML5, CSS3, npm

Todo List App

Aplicación web de gestión de tareas desarrollada con React y Vite que permite crear, editar, completar y eliminar tareas. Incluye filtrado de tareas, confirmaciones visuales con SweetAlert2 y persistencia automática de datos.

Tecnologías: React, Vite, JavaScript, SweetAlert2, HTML5, CSS3, LocalStorage
